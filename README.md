# Projeto-An-lise-de-Vendas-com-Python-Pandas-
Desenvolvi um projeto de análise de dados em Python utilizando Pandas, focado na consolidação e tratamento de múltiplas fontes de dados de vendas.

O projeto simula um cenário real de dados desestruturados, onde implementei um pipeline robusto capaz de:

✔ Ler automaticamente múltiplos arquivos CSV  
✔ Tratar inconsistências como valores nulos, colunas divergentes e erros de formatação  
✔ Padronizar dados textuais, numéricos e temporais  
✔ Criar métricas de negócio como faturamento  
✔ Gerar análises como produtos mais vendidos e performance por região  

O foco principal foi desenvolver um pipeline resiliente, preparado para dados reais, indo além de datasets "limpos".

Tecnologias: Python | Pandas  

Próximos passos: criação de dashboards interativos e geração de insights estratégicos.
